q1-why we use rel in anchor tag, link.
ans-rel(relation)- The rel attribute specifies the relationship between the current document and the linked document.
 Search engines can use this attribute to get more information about a link.

 q2- role of href in anchor tag , link tag
 The href attribute specifies the URL of the page the link goes to.

If the href attribute is not present, the <a> tag will not be a hyperlink.
rel in not mendatory but href is mendatory to mentioned in anchor tag, link tag.
